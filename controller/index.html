How it Works

Start controlling your presentation in less than a minute.

1

Run the Agent

Download and run the SlideMobileAgent.exe on your presentation PC. It securely waits for commands.

Note: Because the file is unsigned, you may see a "Windows protected your PC" warning. Click More info → Run anyway. Our tool is 100% open-source so you can verify it's safe.

Download Agent
2

Scan the QR Code

The Windows Agent will display a QR code and a 6-digit session pin. Point your phone's camera at it to instantly open the web controller.

3

Present Freely

Put your slides (PowerPoint/Keynote) in full screen. Tap the large, touch-optimized Prev/Next buttons on your phone to drive the presentation.
